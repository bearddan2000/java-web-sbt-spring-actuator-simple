# java-web-sbt-spring-actuator-simple

## Description
A POC for actuator in springframework.

## Tech stack
- java
- sbt

## Docker stack
- openjdk:8-jdk-alpine

## To run
`sudo ./install.sh -u`
Available at:
- http://localhost/actuator
- http://localhost/hello
- http://localhost/actuator/info

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
